# 🛒 Shopify Sales & Revenue Analysis Dashboard

## 📌 Project Overview

This project is an interactive **Sales & Revenue Analysis Dashboard** built using **Python, Streamlit, and Plotly**.
It analyzes Shopify trending product data to provide insights into **revenue performance, product trends, and business opportunities**.

The dashboard helps users explore data through **dynamic filters, KPIs, and visualizations**, enabling better decision-making.

---

## 🎯 Key Features

* 📊 **KPI Metrics**

  * Total Revenue
  * Total Units Sold
  * Average Trend Score
  * Top Performing Product

* 🔍 **Interactive Filters**

  * Category-wise filtering
  * Company/Brand filtering
  * Revenue threshold filter
  * Top N product selector

* 📈 **Visualizations**

  * Top Products by Revenue (Bar Chart)
  * Category-wise Revenue Distribution (Donut Chart)
  * Trend vs Revenue Analysis (Bubble Chart)
  * Category Performance Comparison

* 💡 **Business Insights**

  * Automatically generated insights
  * Identification of high-performing products
  * Trend-based opportunity analysis

## 🚀 How to Run the Project

### 1️⃣ Clone the repository

```
git clone https://github.com/BhartiSahu30/Shopify-Sales-and-Revenue-Dashboard.git
```

### 2️⃣ Navigate to project folder

```
cd Shopify-Sales-and-Revenue-Dashboard
```

### 3️⃣ Install dependencies

```
python -m pip install -r requirements.txt
```

### 4️⃣ Run the app

```
python -m streamlit run app.py
```

### 5️⃣ Open in browser

```
http://localhost:8501
```
